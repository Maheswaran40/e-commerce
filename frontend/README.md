# ✅ Authentication APIs
1. POST /auth/register - Register new user

2. POST /auth/login - Login user

3. GET /auth/me - Get current user info

# ✅ Product APIs
1. GET /products - Get all products

2. GET /products/:id - Get single product

3. POST /products - Create product (admin)

4. PUT /products/:id - Update product (admin)

5. DELETE /products/:id - Delete product (admin)

# ✅ Cart APIs
1. GET /cart - Get user cart

2. POST /cart/add - Add item to cart

3. PUT /cart/update - Update cart item quantity

4. DELETE /cart/remove/:productId - Remove item from cart

5. DELETE /cart/clear - Clear entire cart
